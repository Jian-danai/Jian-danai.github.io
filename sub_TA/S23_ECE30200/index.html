CP HINTs

Q1:

You should be familiar with:

  1. the definition of CDF: FX(x) = Pr(X ≤ x)
  2. the properties of inverse function: F(F-1(x)) = x and F-1(F(x)) = x. See https://en.wikipedia.org/wiki/Inverse_function and https://www.math.utah.edu/lectures/math1220/2PostNotes.pdf Page2
  3. the properties of increasing function f(x): a ≤ b \Leftrightarrow f(a) ≤ f(b)
  4. the CDF of the continuous Uniform Distribution between 0 and 1: FU(u) = u when u ∈ (0,1), FU(u) = 1 when u ≥ 1, FU(u) = 0 when u ≤ 0.
